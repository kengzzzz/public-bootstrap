# Compile Proton

- `docker compose run --rm proton-builder`

# Installation

- `mkdir -p ~/.steam/root/compatibilitytools.d/`
- `cp -r ./out/proton_tkg_* ~/.steam/root/compatibilitytools.d/`
- Restart Steam
